## 助记符

- `ldc` ：表示将int、float 或者 String 类型的常量值从常量池中推送至栈顶

- `binpush` : 表示将单字节（-128-127）的常量值推送至栈顶

- `sipush` ：表示将 **短整型常量值**（-32768-32767）推送至栈顶

- `iconst_x`
    - `iconst_m1` ：将**数字 -1**推送到栈顶
    - `iconst_0` ：将**数字 0**推送到栈顶  
        ...
    - `iconst_5` ：将 5 推送到栈顶

- `anewarray` ：创建一个 **引用类型** 的数组，并将其引用值加入栈顶

- `newarray` ：创建一个 **基本类型** 的数组，并将其引用值加入栈顶
