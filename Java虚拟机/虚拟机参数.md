## 虚拟机参数

- **标准参数**

- **-X参数**

- **-XX参数** 

### 标准参数

- `java -help` ：查看标准参数

- `java -D<Name>=<Value>` ：设置系统属性

### -XX

`-XX:+<option>` ：开启 option 选项

`-XX:+<option>` ：关闭 option 选项

`-XX:<option>-<value>` ：将 option 选项的值设置为 value

- `-XX:+TraceClassLoading` ：查看类的加载过程，追踪类的加载过程，并打印

- `-XX:+TraceClassUnloading` ：查看类的卸载过程
