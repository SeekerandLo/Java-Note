### 时间复杂度定义
&emsp;&emsp;在进行算法分析时，语句总的执行次数T(n)，是关于问题规模n的函数，进而分析T(n)随n的变化情况并确定T(n)的数量级。  
&emsp;&emsp;算法的时间复杂度，记作`T(n) = O(f(n))`，表示随问题规模n的增大，算法执行时间的增长率和f(n)的增长率相同，称作时间复杂度，其中f(n)是问题规模n的某个函数。
