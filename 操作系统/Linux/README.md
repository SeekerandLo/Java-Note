## Linux

### 查看端口占用状态

- netstat
    - -t：显示TCP端口
    - -u：显示UDP端口
    - -a：显示全部端口
    - -r：显示路由信息
    - -p：显示PID及程序信息
    - -l: 显示监听的服务端套接字

### 查看进程状态

- ps  process status  
    
    参数
    - -A:显示所有进程，和 -e 相同
    - -a:显示现行终端机下的所有进程，包括其他用户进程
    - -u:以用户为主的进程状态
    - -x:显示没有控制终端的进程，同时显示各个命令的具体路径
    - -f:以树状显示  
    
    例子：   
    - ps -ef：查看所有进程
    - ps -aux：显示所有包含其他使用者的行程
